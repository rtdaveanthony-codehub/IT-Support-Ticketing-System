IT Support Ticketing System (Spiceworks Lab Project)

Designed and simulated an IT Help Desk environment using Spiceworks to manage and resolve technical support tickets, demonstrating real-world IT support workflows and incident management practices. The system was used to log, categorize, prioritize, and track IT issues such as network connectivity failures, VPN access problems, and permission-related access requests. Implemented proper ticket lifecycle handling including ticket creation, troubleshooting, escalation, resolution, and closure documentation in alignment with standard ITIL support processes. Developed skills in incident management, user support, system diagnostics, and service request handling while improving efficiency in tracking and resolving IT issues in a structured support environment.
